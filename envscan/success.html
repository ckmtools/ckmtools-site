Set up the GitHub Action

# .github/workflows/env-check.yml
name: Env Check
on: [pull_request]
jobs:
  validate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: ckmtools/envscan-action@v1
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          license-key: your-email@example.com  # ← replace with your Stripe email

Replace your-email@example.com with the email from your Stripe confirmation. The Action validates this to enable PR comment posting.

What happens next

  • The Action runs on every pull request
  • When new process.env references appear in a diff without a matching .env.example entry, the Action posts an inline PR comment
  • You'll get a Stripe receipt by email — keep it as proof of subscription
